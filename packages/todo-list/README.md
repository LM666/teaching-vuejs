### Tasking


- [ ] 可以通过输入框输入一个 todo
  - 按下回车即可完成一个输入
- [ ] todo

  - [ ] 名称
    - [ ] 为输入框输入的内容
  - [ ] 删除按钮
    - [ ] 点击后，删除当前 todo
  - [ ] 完成按钮
    - [ ] 点击后，当前 todo 的样式画个横线
      - 和普通的 todo 样式区分出来

- [ ] all 按钮，显示所有 todo
- [ ] active 按钮，显示所有处于 active 状态下的所有 todo
- [ ] completed 按钮，显示所有处于 completed 状态下的所有 todo

- [ ] Clear completed
  - 清空当前选中的所有处于 completed 状态下的 todo
- [ ] 显示当前处于 active 状态的 todo 数量

* [ ] 可保存
  - 保存到 localStorage
